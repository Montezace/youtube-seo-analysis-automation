# YouTube Channel Analyzer & SEO Tool

A Python-based automation tool to scrape top-performing videos from any YouTube channel and perform deep SEO analysis on specific videos using OpenAI.

## Features
- **Channel Top Hits:** Extract the top 10, 20, or 30 most viewed videos from any channel.
- **Automated Summaries:** Generates a text report of video descriptions and links.
- **AI SEO Analysis:** Downloads video transcripts and uses GPT to analyze keywords, hooks, and SEO optimization methods.

## Installation
1. Clone the repo: `git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Set your API Keys in the script or as environment variables.
